Elder Scroll: The Chosen One

I am proud to say that this is my creation for Mid Semester Assesment. I went through a lot with this assignment and I really liked it! This is a asteroid-like game.

We are to make a 2D game with several rules such as:

  • An asteroid continuously moves in any one random direction (not just up, down, left, and right). It also rotates continuously
  • Medium asteroids move faster than big asteroids, and small asteroids move faster than both big and medium
  • The bullet travels forward in the same direction that the spaceship was facing when the fire key was pressed
  • When a bullet hits an asteroid, the asteroid is destroyed. Nothing happens when 2 asteroids collide
  • There should be a sound effect every time the spaceship fires a bullet

The theme that I chose is Elder Scroll, one of my favourite game of all time.

Scenes

The title screen is made up of 4 elements.

  • The title of the game.
  • Directions for the user
  • Background
  • A button to refer the user to the next scene

This screen is where the user will prepare themselve before clicking the button. If this screen does not exist and the game just starts without a warning, boy that would be unpleasant!

The main screen is made up of 4 categorized elements.

  • Protagonist
  • Enemies
  • Power Ups
  • Information Labels

This screen is where all the action starts! Enemies will fly around, the boss will randomly pop up and start shooting.

The game over screen is made up of 4 elements.

  • The final score
  • Background
  • Directions for the user
  • A button to refer the user back to the title screen

This screen will pop up when the user has no more lives left. The final score will be shown at the top left of the screen.
